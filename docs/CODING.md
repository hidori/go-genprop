# コーディング規約

## コメント

- コメントは英語で記述する
- パブリックスコープの godoc コメントを必須とする
- 実装とコメントに乖離があったらコメントを修正する

## 命名規則

- メンバメソッドのレシーバ名は型名の頭文字を使用する
  - `User` 型のメンバメソッドのレシーバ名は `u`
- **プロジェクト固有**: `public/generator/Generator` が生成するコードのレシーバ名は `t` 固定
  - 生成されるすべてのメソッドで一貫したレシーバ名を使用するため

## エラー

- エラー返却時は `errors.WithStack()` または `errors.Wrap()`/`errors.Wrapf()` でスタックトレースを付与する
  - コンテキスト情報が必要な場合は `errors.Wrap()`/`errors.Wrapf()` を使用する
  - コンテキスト情報が不要な場合は `errors.WithStack()` を使用する

## パッケージ

- base.go にパッケージの godoc コメントを記述する
